## Docker 使用指南

你可以直接拉取官方镜像运行：

```bash
# 拉取最新镜像
docker pull langya466/geek2blogserver:latest

# 运行容器
docker run -d -p 8080:8080 langya466/geek2blogserver:latest

```
> 💡 **Tip:** 如需绑定域名请使用反向代理
